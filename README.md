# spaXes - Farcaster Mini App

A multi-page Farcaster mini app for discovering live/past/future Twitter Spaces hosted by Farcaster users, and a powerful Spaces Clip Tool for hosts.

## Features
- See live Twitter Spaces
- See which Farcaster users are hosting Spaces
- View all past/future Spaces for a user
- Safely link Farcaster and Twitter accounts
- Clip Tool: Mark and segment Spaces with speaker/time tracking (local storage)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Environment Variables
- `NEY_API_KEY`: Your Neynar API key
- `TWITTER_BEARER_TOKEN`: Your Twitter Bearer Token

## Project Structure
- `/pages` - App routes (multi-page)
- `/components` - Shared UI components
- `/lib` - Utility functions (API, auth, etc)

## Deployment
Deploy instantly to Vercel for HTTPS and Farcaster Mini App compatibility.
